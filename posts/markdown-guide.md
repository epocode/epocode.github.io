# Markdown 写作指南

Markdown 是一种轻量级标记语言，让你可以使用简单的纯文本格式来编写文档。本文介绍常用的 Markdown 语法。

## 标题

使用 `#` 符号来创建标题，`#` 的数量表示标题级别：

```markdown
# 一级标题
## 二级标题
### 三级标题
```

## 文本格式

- **粗体**：`**粗体文字**`
- *斜体*：`*斜体文字*`
- ~~删除线~~：`~~删除线~~`
- `行内代码`：`` `代码` ``

## 列表

### 无序列表

```markdown
- 项目一
- 项目二
  - 子项目
```

### 有序列表

```markdown
1. 第一步
2. 第二步
3. 第三步
```

## 链接和图片

```markdown
[链接文字](https://example.com)
![图片描述](image.jpg)
```

## 引用

使用 `>` 创建引用块：

> 这是一段引用文字。
> 可以有多行。

## 代码块

使用三个反引号创建代码块，可以指定语言：

```python
def hello():
    print("Hello, World!")
```

```javascript
function greet(name) {
    console.log(`Hello, ${name}!`);
}
```

## 表格

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| A1  | B1  | C1  |
| A2  | B2  | C2  |
```

效果：

| 功能 | 语法 | 示例 |
|------|------|------|
| 粗体 | `**文字**` | **粗体** |
| 斜体 | `*文字*` | *斜体* |
| 代码 | `` `代码` `` | `code` |

## 写作建议

1. **保持简洁** - 每段话表达一个主要观点
2. **善用标题** - 帮助读者快速浏览
3. **代码高亮** - 技术文章要标注代码语言
4. **添加图片** - 一图胜千言

> 好的文章不在于写得多长，而在于表达得多清晰。
