# 欢迎来到我的博客

欢迎来到我的个人博客！这是一个使用纯 HTML、CSS 和 JavaScript 构建的静态博客网站，可以轻松部署到 GitHub Pages。

## 关于这个博客

这个博客的设计目标是简洁、现代、易用。它包含了以下特性：

- **响应式设计** - 支持移动端和桌面端
- **深色/浅色主题** - 自动保存主题偏好
- **Markdown 支持** - 只需编写 md 文件即可发布文章
- **简洁美观** - 传统中式风格的界面设计

## 如何添加新文章

添加新文章非常简单，只需两步：

1. 在 `posts` 目录下创建一个新的 `.md` 文件
2. 在 `posts.js` 中添加文章的索引信息

```javascript
{
    id: 4,  // 唯一 ID
    title: "文章标题",
    date: "2026-01-15",
    file: "posts/your-article.md",
    excerpt: "文章摘要",
    tags: ["标签1", "标签2"],
    author: "作者名"
}
```
